Here are some of my projects:

For our thesis, Me and my groupmates developed a browser-based system called HardwareStorePlus. I was tasked on developing its front-end but from time-to-time
I helped in developing its back-end. It has two access level, admin and employees, wherein these two access has difference functions. The admin level can add, update,
delete, and view the employees' and product's information, take customer orders, and could generate sales report by automatically creating its excel file. And as for the employee level, it is limited in viewing of products, and take customer orders.

We were tasked to develop a project for our finals using django framework with Python, HTML, and CSS as the programming languages. The project included the
CRUD functions, and uses an API that we developed. The name of the project is GoMovie because it manages the movies that are recorded in the system.

We were tasked to develop a "Adventurer vs Monsters" game. This game is a rock, paper, scissors based game where the user get to pick which adventurer and
monster will fight. It decides the winner by comparing the strength of these two.

We were tasked to create our own portfolio using HTML and CSS.
